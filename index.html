1. Objetivos del Proyecto

Este proyecto final de la asignatura de Sistemas Operativos tiene como objetivo aplicar de manera práctica los conceptos estudiados durante el curso. Consiste en la instalación y configuración de distribuciones Linux (en modo gráfico y consola), el uso de scripts Bash para automatización de tareas, y la implementación de tecnologías modernas como Docker y Kubernetes para la creación y despliegue de contenedores. Además, se documenta todo el proceso mediante capturas, videos y un sitio web como evidencia completa del trabajo realizado.
